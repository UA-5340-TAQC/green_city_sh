**Test Case ID:** TC-04  
**Title:** Verify that search displays events filtered by the keyword
**Related Requirement/User Story:** User story #Events-Search  
**Author:** Voloshyn Volodymyr  
**Priority:** High

**Preconditions:**

1. User is authorized on the GreenCity website.
2. User is on the Events page: `https://www.greencity.cx.ua/#/greenCity/events`.
3. Events containing the keyword in their title.

**Test Steps:**
| Step # | Step Description | Test Data | Expected Result |
|:------:|:----------------|:----------|:---------------|
| 1 | Click on the search icon inside the events container | `span.search-img` | Search input field is displayed and focused |
| 2 | Enter the search keyword into the input field | Keyword (e.g., "2026") | The keyword is correctly displayed in the field |
| 3 | Wait for the search results to update | | The list updates dynamically or after pressing Enter |
| 4 | Observe the event list and count of found items | | Only events containing the keyword are displayed; search results are relevant |

**Postconditions:**

1. Search results remain visible.
2. Search can be cleared by removing the text or clicking the Search close button.

- **Browser:** Chromium, Firefox, WebKit
- **Version:** Latest
